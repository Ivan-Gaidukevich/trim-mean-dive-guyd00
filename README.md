## Усеченное среднее

Усечённое среднее вычисляется путем отбрасывания `p` наименьших `p` наибольших значений из упорядоченной выборки. Пусть даны отсортированные значения:

**Например, в международных состязаниях (чемпионатах) по прыжкам в воду:**

- оценки выставляют `5` судей
- самый верхний и нижний балл `(от 0 до 10)` судей отбрасываются
- итоговый балл - среднеарифметический балл трех оставшихся судей.

Такой подход не дает одному судье манипулировать баллом, возможно, чтобы оказать содействие спортсмену из своей страны.

**На Олимпийских играх и Кубках мира прыжки оценивают по-другому.**

- оценки выставляют `7` судей
- прыжок оценивается по `10-балльной` системе:
- от `0` (низшая оценка) до `10` (высшая).
- При выведении общей оценки за прыжок высшая и низшая оценки отбрасываются
- Оставшиеся оценки суммируются и умножаются на коэффициент сложности прыжка `(от 1,2 до 3,6)`.
- Полученный результат умножается на `0,6`.
